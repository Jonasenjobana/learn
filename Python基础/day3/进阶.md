## 1. 继承是什么

继承的意思是：子类可以直接复用父类的属性和方法，再在此基础上增加或修改自己的内容。

比如：

- `Animal` 是父类
- `Dog`、`Cat` 是子类

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name} 发出声音")


class Dog(Animal):
    pass


dog = Dog("小黑")
dog.speak()
```

这里 `Dog(Animal)` 的意思是：`Dog` 继承 `Animal`。

## 2. 子类重写父类方法

子类可以保留父类的结构，但把某个方法改成自己的版本。

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name} 发出声音")


class Dog(Animal):
    def speak(self):
        print(f"{self.name} 汪汪叫")


class Cat(Animal):
    def speak(self):
        print(f"{self.name} 喵喵叫")


dog = Dog("小黑")
cat = Cat("小白")

dog.speak()
cat.speak()
```

这叫“方法重写”。

## 3. `super()` 的作用

`super()` 用来调用父类的方法，最常见是在子类的 `__init__` 里先把父类初始化好。

```python
class Animal:
    def __init__(self, name):
        self.name = name


class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)
        self.breed = breed


dog = Dog("小黑", "柯基")
print(dog.name)
print(dog.breed)
```

### 为什么要用 `super()`

- 复用父类逻辑
- 少写重复代码
- 子类只补自己新增的部分

## 4. 继承适合什么场景

继承适合“是一个”的关系。

比如：

- `Dog` 是一个 `Animal`
- `Student` 是一个 `Person`
- `AdminUser` 是一个 `User`

如果不是“是一个”的关系，通常不要硬继承。

## 5. 组合通常比继承更灵活

如果是“有一个”的关系，更适合组合。

比如：

- `Car` 有一个 `Engine`
- `Order` 有一个 `User`
- `Student` 有一个 `Address`

```python
class Engine:
    def start(self):
        print("发动机启动")


class Car:
    def __init__(self, brand):
        self.brand = brand
        self.engine = Engine()

    def start(self):
        self.engine.start()
        print(f"{self.brand} 开动了")
```

很多项目里，组合会比继承更清晰。

## 6. 类方法 `@classmethod`

类方法的第一个参数是 `cls`，表示类本身。

```python
class Student:
    school = "Python 学习班"

    def __init__(self, name, score):
        self.name = name
        self.score = score

    @classmethod
    def from_dict(cls, data):
        return cls(data["name"], data["score"])


student = Student.from_dict({"name": "小明", "score": 88})
print(student.name, student.score)
```

类方法常用于：

- 通过别的数据结构创建对象
- 修改类级别的数据
- 提供替代构造方式

## 7. 静态方法 `@staticmethod`

静态方法既不需要 `self`，也不需要 `cls`。

它更像“放在类里的普通工具函数”。

```python
class MathTool:
    @staticmethod
    def add(a, b):
        return a + b


print(MathTool.add(3, 5))
```

什么时候用静态方法：

- 这个函数和类有关
- 但它不依赖对象状态
- 也不依赖类状态

## 8. 属性控制：`property`

`property` 可以把“方法”包装成“属性”的访问方式。

```python
class Student:
    def __init__(self, score):
        self._score = score

    @property
    def score(self):
        return self._score

    @score.setter
    def score(self, value):
        if value < 0:
            value = 0
        self._score = value


student = Student(88)
print(student.score)
student.score = -10
print(student.score)
```

这在需要校验数据时很有用。

## 9. 常见魔术方法

魔术方法是以双下划线开头和结尾的方法。

### `__str__` 和 `__repr__`

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def __str__(self):
        return f"Student(name={self.name}, score={self.score})"


student = Student("小明", 88)
print(student)
```

`__str__` 用于更友好的字符串显示。

### `__len__`

```python
class Classroom:
    def __init__(self, students):
        self.students = students

    def __len__(self):
        return len(self.students)


classroom = Classroom(["小明", "小红", "小刚"])
print(len(classroom))
```

### `__add__`

```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)


p1 = Point(1, 2)
p2 = Point(3, 4)
p3 = p1 + p2
print(p3.x, p3.y)
```

这类方法让对象可以参与更自然的运算。

## 10. 多继承

Python 支持多继承：

```python
class A:
    pass


class B:
    pass


class C(A, B):
    pass
```

但在入门阶段，你先把它理解成“能用”，不要急着“多用”。

多继承会让代码复杂，方法查找顺序也更绕。

很多情况下，组合会更清楚。

## 11. `isinstance` 和 `issubclass`

这两个函数在判断类型关系时很常用。

```python
class Animal:
    pass


class Dog(Animal):
    pass


dog = Dog()

print(isinstance(dog, Dog))
print(isinstance(dog, Animal))
print(issubclass(Dog, Animal))
```

## 12. 一个完整例子

```python
class Person:
    def __init__(self, name):
        self.name = name

    def info(self):
        return f"姓名：{self.name}"


class Student(Person):
    school = "Python 学习班"

    def __init__(self, name, score):
        super().__init__(name)
        self.score = score

    def is_passed(self):
        return self.score >= 60

    def info(self):
        status = "及格" if self.is_passed() else "不及格"
        return f"{super().info()}，分数：{self.score}，{status}"


student = Student("小明", 88)
print(student.info())
```

这里同时用了：

- 继承
- `super()`
- 方法重写
- 类属性

你现在阶段最重要的是：

- 会写一个简单类

- 知道什么时候用组合
- 看懂 `classmethod`、`staticmethod`、`property`

不用一开始就把所有魔术方法背完，先建立“类能组织数据和行为”的感觉。
