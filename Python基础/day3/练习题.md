# Day3 练习题：类与对象

这一阶段专门练 Python 的类。

## 题目 1：Student 类

定义一个 `Student` 类，要求：

- 有 `name` 和 `score` 两个属性
- 有 `show_info()` 方法，输出学生信息
- 有 `is_passed()` 方法，判断是否及格

要求：

- 及格线按 `60` 分算
- 输出格式清楚

## 题目 2：Book 类

定义一个 `Book` 类，要求：

- 有 `title`
- 有 `author`
- 有 `price`
- 有 `show_info()` 方法

然后创建 3 本书，放进列表里遍历输出。

## 题目 3：Car 类

定义一个 `Car` 类，要求：

- 有 `brand`
- 有 `speed`
- 有 `accelerate()` 方法，可以让速度增加
- 有 `brake()` 方法，可以让速度减少
- 有 `show_speed()` 方法输出当前速度

## 题目 4：类属性

在 `Student` 类里加一个类属性 `school = "Python 学习班"`。

要求：

- 所有学生都能访问这个属性
- `show_info()` 时一起输出学校名

## 题目 5：修改对象状态

给 `Student` 类增加一个 `update_score(new_score)` 方法。

要求：

- 调用后能修改分数
- 再次调用 `show_info()` 时能看到新分数

## 题目 6：对象列表

创建 3 个学生对象，放进一个列表。

要求：

- 使用 `for` 循环遍历
- 每个对象调用 `show_info()`

## 题目 7：字典和类对比

先用字典表示一个学生：

```python
student = {"name": "小明", "score": 88}
```

再用类表示同样的数据。

要求：

- 你能说出两种写法的差别
- 你能理解什么时候适合用字典，什么时候适合用类

## 题目 8：综合小练习

定义一个 `Product` 类，要求：

- 有 `name`
- 有 `price`
- 有 `count`
- 有 `total_price()` 方法，返回总价
- 有 `show_info()` 方法，输出商品信息

## 检查标准

如果你做完了，应该能解释：

- `class` 是什么
- `self` 是什么
- `__init__` 是干什么的
- 实例属性和类属性的区别
- 为什么类比字典更适合表达“有数据也有行为”的对象
