# Day5 练习题：文件读写、JSON 与 CSV

## 题目 1：写入文本

写一个程序，把下面内容写入 `note.txt`：

```text
今天学习 Python 文件读写
```

要求：

- 使用 `with open`
- 使用 `encoding="utf-8"`

## 题目 2：追加文本

在 `note.txt` 末尾追加一行：

```text
继续学习 JSON 和 CSV
```

要求：

- 使用追加模式 `a`

## 题目 3：读取文本

读取 `note.txt` 的全部内容并打印。

要求：

- 使用 `read()`
- 处理文件不存在的情况

## 题目 4：逐行读取

逐行读取 `note.txt`。

要求：

- 使用 `for line in file`
- 输出时去掉行尾换行

## 题目 5：写入 JSON

创建一个学生字典：

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 88
}
```

写入 `student.json`。

要求：

- 使用 `json.dump`
- 中文不能被转义
- 文件格式要方便阅读

## 题目 6：读取 JSON

读取 `student.json` 并输出学生姓名和分数。

要求：

- 使用 `json.load`

## 题目 7：写入 CSV

给定：

```python
students = [
    {"name": "小明", "score": 88},
    {"name": "小红", "score": 95},
    {"name": "小刚", "score": 76},
]
```

写入 `students.csv`。

要求：

- 使用 `csv.DictWriter`
- 写入表头
- Windows 下注意 `newline=""`

## 题目 8：读取 CSV

读取 `students.csv` 并输出所有学生信息。

要求：

- 使用 `csv.DictReader`
- 注意分数读出来是字符串

## 题目 9：计算平均分

读取 `students.csv`，计算平均分。

要求：

- 把 `score` 转成整数
- 输出平均分

## 题目 10：综合练习

写一个程序：

- 先创建一个学生列表
- 保存到 JSON 文件
- 再保存到 CSV 文件
- 从 CSV 文件中读回来
- 计算最高分学生

## 检查标准

做完后你应该能解释：

- `r`、`w`、`a` 的区别
- 为什么推荐 `with open`
- JSON 和 CSV 适合什么场景
- `json.dump` 和 `json.load` 的区别
- `csv.DictWriter` 和 `csv.DictReader` 的作用
- CSV 读出来的数据为什么需要类型转换
