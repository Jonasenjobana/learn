# Python 数据类型与类

## 本文目标

这份文档重点讲两个内容：

- Python 里常见的数据类型是什么，分别适合存什么数据。
- Python 里的类是什么，为什么后面学习 AI 工具时经常会看到类。

学习建议：先理解概念，再跟着示例敲代码。不要一开始就背所有方法，能看懂、会查、会改，才是入门阶段最重要的。

## 1. 什么是数据类型

程序本质上是在处理数据。不同数据有不同特点，比如：

- 年龄是数字。
- 姓名是文本。
- 是否登录是“是/否”。
- 一组分数需要放在一起。
- 一个学生有姓名、年龄、成绩等多项信息。

Python 会用不同的数据类型来表示这些内容。

常见数据类型：

| 类型 | Python 写法 | 适合表示 |
| --- | --- | --- |
| 整数 | `int` | 年龄、数量、次数 |
| 小数 | `float` | 身高、价格、概率、损失值 |
| 字符串 | `str` | 姓名、文本、路径 |
| 布尔值 | `bool` | 真/假、是/否 |
| 空值 | `NoneType` | 暂时没有值 |
| 列表 | `list` | 一组有顺序的数据 |
| 元组 | `tuple` | 一组不希望被修改的数据 |
| 字典 | `dict` | 键值对数据 |
| 集合 | `set` | 不重复的数据集合 |

可以用 `type()` 查看变量类型：

```python
age = 18
name = "小明"
scores = [88, 92, 76]

print(type(age))
print(type(name))
print(type(scores))
```

输出大概是：

```text
<class 'int'>
<class 'str'>
<class 'list'>
```

这里的 `class` 先不用害怕，它表示这些类型本质上也是 Python 里的“类”。

## 2. 数字类型：int 和 float

### 2.1 int 整数

`int` 用来表示整数。

```python
age = 18
count = 100
temperature = -3
```

常见运算：

```python
a = 10
b = 3

print(a + b)  # 加法，13
print(a - b)  # 减法，7
print(a * b)  # 乘法，30
print(a / b)  # 普通除法，3.333...
print(a // b) # 整除，3
print(a % b)  # 取余，1
print(a ** b) # 次方，1000
```

### 2.2 float 小数

`float` 用来表示小数。

```python
height = 1.75
price = 19.9
loss = 0.253
```

在 AI 学习中，小数很常见，比如：

- 概率：`0.87`
- 学习率：`0.001`
- 模型损失值：`0.3521`
- 准确率：`0.94`

注意：计算机里的小数可能有精度误差。

```python
print(0.1 + 0.2)
```

结果可能是：

```text
0.30000000000000004
```

这是正常现象。入门阶段知道有这个问题即可。

## 3. 字符串 str

字符串用来表示文本，用单引号或双引号都可以。

```python
name = "小明"
city = '上海'
```

多行文本可以用三个引号：

```python
message = """
今天学习 Python。
明天继续学习数据类型。
"""
```

### 3.1 字符串拼接

```python
first_name = "张"
last_name = "三"

print(first_name + last_name)
```

注意：字符串不能直接和数字相加。

```python
age = 18

print("年龄：" + str(age))
```

### 3.2 f-string 格式化

更推荐使用 f-string：

```python
name = "小明"
age = 18

print(f"我叫{name}，今年{age}岁")
```

这在后面写日志、输出模型结果、拼接文件路径时都很常用。

### 3.3 字符串常用方法

```python
text = "  Python is good  "

print(text.strip())       # 去掉两边空格
print(text.lower())       # 转小写
print(text.upper())       # 转大写
print(text.replace("good", "great"))
print(text.split())       # 按空格切分成列表
```

示例：统计一句话里有多少个单词。

```python
sentence = "Python is useful for AI"
words = sentence.split()

print(words)
print(len(words))
```

## 4. 布尔值 bool

布尔值只有两个：

```python
is_login = True
is_admin = False
```

常用于条件判断：

```python
score = 85

if score >= 60:
    print("及格")
else:
    print("不及格")
```

比较运算会得到布尔值：

```python
print(10 > 3)
print(10 == 3)
print("Python" == "Python")
```

常见逻辑运算：

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("可以入场")
```

## 5. 空值 None

`None` 表示“没有值”或“暂时为空”。

```python
result = None
```

它和 `0`、空字符串 `""` 不一样。

```python
score = None

if score is None:
    print("还没有录入成绩")
```

判断是否为 `None` 时，推荐使用：

```python
value is None
value is not None
```

## 6. 列表 list

列表用来保存一组有顺序的数据。

```python
scores = [88, 92, 76, 95]
names = ["小明", "小红", "小刚"]
mixed = ["小明", 18, True]
```

虽然列表可以混合存不同类型，但实际开发中，通常会让一个列表里的数据类型尽量一致。

### 6.1 访问列表元素

```python
names = ["小明", "小红", "小刚"]

print(names[0])
print(names[1])
print(names[-1])
```

索引从 `0` 开始：

```text
索引：  0      1      2
数据： 小明   小红   小刚
```

### 6.2 修改列表

```python
names = ["小明", "小红"]

names.append("小刚")
names[0] = "小王"
names.remove("小红")

print(names)
```

常用方法：

| 方法 | 作用 |
| --- | --- |
| `append(value)` | 末尾添加元素 |
| `insert(index, value)` | 指定位置插入 |
| `remove(value)` | 删除指定值 |
| `pop()` | 删除并返回最后一个元素 |
| `sort()` | 排序 |
| `reverse()` | 反转 |

### 6.3 遍历列表

```python
scores = [88, 92, 76]

for score in scores:
    print(score)
```

如果需要索引：

```python
names = ["小明", "小红", "小刚"]

for index, name in enumerate(names):
    print(index, name)
```

### 6.4 列表切片

```python
numbers = [0, 1, 2, 3, 4, 5]

print(numbers[1:4])
print(numbers[:3])
print(numbers[3:])
print(numbers[::2])
```

切片常用于取一部分数据。后面处理训练数据时，也会经常看到类似思想。

## 7. 元组 tuple

元组和列表很像，也是一组有顺序的数据，但元组创建后不能修改。

```python
point = (10, 20)
rgb = (255, 0, 0)
```

适合表示不希望被改动的数据，比如：

- 坐标
- 颜色
- 固定配置
- 函数返回的多个值

```python
def get_user():
    return "小明", 18

name, age = get_user()
print(name, age)
```

注意：只有一个元素的元组要加逗号。

```python
value = (10,)
```

## 8. 字典 dict

字典用来保存键值对。

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 88,
}
```

可以理解成：

```text
name  -> 小明
age   -> 18
score -> 88
```

### 8.1 读取和修改字典

```python
student = {
    "name": "小明",
    "age": 18,
}

print(student["name"])

student["score"] = 88
student["age"] = 19

print(student)
```

如果不确定键是否存在，推荐用 `get()`：

```python
student = {"name": "小明"}

print(student.get("score"))
print(student.get("score", 0))
```

### 8.2 遍历字典

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 88,
}

for key, value in student.items():
    print(key, value)
```

### 8.3 列表和字典组合

实际开发中，经常会把多个字典放进列表。

```python
students = [
    {"name": "小明", "score": 88},
    {"name": "小红", "score": 95},
    {"name": "小刚", "score": 76},
]

for student in students:
    print(student["name"], student["score"])
```

这种结构很重要，因为后面处理数据集时，经常会遇到“一条数据是一个字典，多条数据组成一个列表”的形式。

## 9. 集合 set

集合用于保存不重复的数据。

```python
tags = {"AI", "Python", "AI", "深度学习"}

print(tags)
```

重复的 `"AI"` 只会保留一份。

常见用途：

- 去重
- 判断某个元素是否存在
- 求交集、并集、差集

```python
a = {"Python", "AI", "前端"}
b = {"Python", "数据分析", "AI"}

print(a & b)  # 交集
print(a | b)  # 并集
print(a - b)  # 差集
```

列表去重：

```python
numbers = [1, 2, 2, 3, 3, 3]
unique_numbers = list(set(numbers))

print(unique_numbers)
```

注意：集合本身没有固定顺序。

## 10. 可变类型与不可变类型

这是 Python 里很重要的概念。

### 10.1 不可变类型

不可变类型创建后，值本身不能被原地修改。

常见不可变类型：

- `int`
- `float`
- `str`
- `bool`
- `tuple`
- `NoneType`

示例：

```python
name = "Python"
name = name + " AI"
print(name)
```

看起来字符串被修改了，但实际是创建了一个新字符串，再让 `name` 指向新字符串。

### 10.2 可变类型

可变类型可以在原对象上修改内容。

常见可变类型：

- `list`
- `dict`
- `set`

示例：

```python
numbers = [1, 2, 3]
numbers.append(4)

print(numbers)
```

### 10.3 为什么要关心这个

看下面这个例子：

```python
a = [1, 2, 3]
b = a

b.append(4)

print(a)
print(b)
```

输出：

```text
[1, 2, 3, 4]
[1, 2, 3, 4]
```

原因是 `a` 和 `b` 指向同一个列表。通过 `b` 修改列表，`a` 看到的内容也变了。

如果想复制一个新列表，可以这样：

```python
a = [1, 2, 3]
b = a.copy()

b.append(4)

print(a)
print(b)
```

## 11. 类型转换

常见类型转换：

```python
age_text = "18"
age = int(age_text)

price_text = "19.9"
price = float(price_text)

number = 100
text = str(number)
```

输入内容默认是字符串：

```python
age = input("请输入年龄：")
print(type(age))
```

如果要做数学计算，需要转换：

```python
age = int(input("请输入年龄："))
print(age + 1)
```

列表、元组、集合之间也可以转换：

```python
numbers = [1, 2, 2, 3]

print(tuple(numbers))
print(set(numbers))
print(list(set(numbers)))
```

## 12. 什么是类

类可以理解成“模板”。

对象可以理解成“根据模板创建出来的具体东西”。

例如，`学生` 是一个类，`小明` 是一个具体学生对象。

一个学生通常有：

- 姓名
- 年龄
- 分数
- 自我介绍的行为

用普通字典可以这样表示：

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 88,
}
```

用类可以这样表示：

```python
class Student:
    def __init__(self, name, age, score):
        self.name = name
        self.age = age
        self.score = score

    def introduce(self):
        print(f"我叫{self.name}，今年{self.age}岁，分数是{self.score}")


student = Student("小明", 18, 88)
student.introduce()
```

## 13. 类、对象、属性、方法

### 13.1 类 class

类是模板。

```python
class Student:
    pass
```

### 13.2 对象 object

对象是根据类创建出来的具体实例。

```python
student1 = Student()
student2 = Student()
```

`student1` 和 `student2` 都是 `Student` 类创建出来的对象，但它们是两个不同对象。

### 13.3 属性 attribute

属性是对象身上保存的数据。

```python
class Student:
    def __init__(self, name):
        self.name = name


student = Student("小明")
print(student.name)
```

这里的 `name` 就是属性。

### 13.4 方法 method

方法是对象能做的事情，本质上是定义在类里的函数。

```python
class Student:
    def __init__(self, name):
        self.name = name

    def say_hello(self):
        print(f"你好，我是{self.name}")


student = Student("小明")
student.say_hello()
```

## 14. __init__ 和 self

### 14.1 __init__

`__init__` 是初始化方法。创建对象时，它会自动执行。

```python
class Student:
    def __init__(self, name):
        self.name = name


student = Student("小明")
```

执行 `Student("小明")` 时，Python 会自动调用 `__init__`，并把 `"小明"` 传给 `name`。

### 14.2 self

`self` 表示当前对象自己。

```python
class Student:
    def __init__(self, name):
        self.name = name

    def introduce(self):
        print(self.name)
```

如果创建两个对象：

```python
student1 = Student("小明")
student2 = Student("小红")

student1.introduce()
student2.introduce()
```

`student1.introduce()` 里的 `self` 指向 `student1`。

`student2.introduce()` 里的 `self` 指向 `student2`。

## 15. 类属性与实例属性

### 15.1 实例属性

实例属性属于某一个对象。

```python
class Student:
    def __init__(self, name):
        self.name = name


student1 = Student("小明")
student2 = Student("小红")

print(student1.name)
print(student2.name)
```

`student1.name` 和 `student2.name` 可以不同。

### 15.2 类属性

类属性属于类本身，通常用于所有对象共享的数据。

```python
class Student:
    school = "AI 学习班"

    def __init__(self, name):
        self.name = name


student1 = Student("小明")
student2 = Student("小红")

print(student1.school)
print(student2.school)
print(Student.school)
```

如果每个学生都属于同一个学校，可以使用类属性。

## 16. 类方法里修改对象状态

对象的方法可以修改自己的属性。

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def update_score(self, new_score):
        self.score = new_score

    def is_passed(self):
        return self.score >= 60


student = Student("小明", 58)
print(student.is_passed())

student.update_score(80)
print(student.is_passed())
```

这里：

- `update_score` 用来修改分数。
- `is_passed` 用来判断是否及格。

## 17. 什么时候用字典，什么时候用类

入门阶段可以这样判断：

如果只是临时保存一组数据，用字典。

```python
student = {
    "name": "小明",
    "score": 88,
}
```

如果数据和行为经常一起出现，用类。

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def is_passed(self):
        return self.score >= 60
```

简单对比：

| 场景 | 推荐 |
| --- | --- |
| 读取接口返回的数据 | 字典 |
| 保存配置 | 字典 |
| 表示一个有行为的对象 | 类 |
| 游戏角色、学生、订单、模型等实体 | 类 |
| 入门练习小脚本 | 先用字典 |
| 代码逐渐复杂 | 考虑用类 |

## 18. 类和 AI 学习的关系

后面学习 AI 时，经常会看到类。

例如 PyTorch 里定义神经网络，常见写法类似：

```python
class MyModel:
    def __init__(self):
        pass

    def forward(self, x):
        pass
```

真实 PyTorch 代码会更复杂，但核心思想仍然是：

- 用类表示一个模型。
- 用属性保存模型的参数和层。
- 用方法定义模型如何处理输入数据。

所以现在不需要把面向对象学得非常深，但至少要能看懂：

- `class` 是定义类。
- `__init__` 是初始化。
- `self.xxx` 是对象自己的属性。
- `对象.方法()` 是调用对象的行为。

## 19. 综合示例：学生成绩管理

下面这个例子把数据类型和类结合起来。

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def is_passed(self):
        return self.score >= 60

    def show_info(self):
        status = "及格" if self.is_passed() else "不及格"
        print(f"{self.name}：{self.score} 分，{status}")


students = [
    Student("小明", 88),
    Student("小红", 95),
    Student("小刚", 52),
]

total_score = 0

for student in students:
    student.show_info()
    total_score += student.score

average_score = total_score / len(students)
print(f"平均分：{average_score}")
```

这个例子里：

- `Student` 是类。
- `students` 是列表。
- 列表里每个元素都是一个 `Student` 对象。
- `name` 和 `score` 是实例属性。
- `is_passed` 和 `show_info` 是方法。
- `total_score` 和 `average_score` 是数字。
- 输出文字使用了字符串格式化。
