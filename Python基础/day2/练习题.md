# Day2 练习题：Python 速通（JavaScript 基础版）

这一天不再重复练最基础的条件和循环概念，而是专门练 Python 和 JavaScript 的差异点。

## 题目 1：输入和类型转换

输入一个年龄，输出：

```text
你明年将是 X 岁
```

要求：

- `input()` 后必须转成整数
- 使用 `f-string`

## 题目 2：真值判断

判断下面三个值在 Python 里分别是 `True` 还是 `False`：

- `""`
- `"hello"`
- `0`

要求：

- 写成程序输出结果
- 体会 Python 的 truthy / falsy 规则

## 题目 3：None 判断

定义一个变量 `result = None`，判断它是不是空值。

要求：

- 用 `is None`
- 不要用 `== None`

## 题目 4：切片

给定列表：

```python
nums = [10, 20, 30, 40, 50, 60]
```

请输出：

- 前 3 个元素
- 从第 2 个到第 4 个元素
- 每隔 2 个取一个元素

## 题目 5：交换变量

给定：

```python
a = 100
b = 200
```

请交换它们的值。

要求：

- 不能使用临时变量
- 使用 Python 的解包写法

## 题目 6：列表推导式

生成 1 到 20 中所有偶数，放到一个列表里。

要求：

- 使用列表推导式
- 不要用普通 `for` 循环手写结果

## 题目 7：字典遍历

给定字典：

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 88
}
```

请输出每个键和值。

要求：

- 使用 `items()`
- 输出格式清楚

## 题目 8：默认值读取

给定字典：

```python
user = {"name": "Tom"}
```

请读取 `age`，如果没有这个键，就返回 `18`。

要求：

- 使用 `get()`

## 题目 9：enumerate 和 zip

给定两个列表：

```python
names = ["小明", "小红", "小刚"]
scores = [88, 95, 76]
```

完成以下任务：

- 用 `enumerate` 输出序号和名字
- 用 `zip` 输出姓名和分数的组合

## 题目 10：可变对象测试

执行下面这段逻辑并观察结果：

```python
a = [1, 2, 3]
b = a
b.append(4)
```

请解释：

- 为什么 `a` 也变了
- 如果想复制一份新列表，应该怎么做

## 题目 11：循环风格

写一个程序，输出 1 到 20。

要求：

- 每 5 个数字换一行
- 使用 `for`
- 尽量写得像 Python 风格

## 题目 12：综合小练习

给定一个列表：

```python
scores = [88, 92, 76, 95, 67, 100]
```

完成以下任务：

- 找出所有大于 80 的分数
- 求平均分
- 输出最高分和最低分

要求：

- 至少使用一次列表推导式
- 至少使用一次内置函数

## 检查标准

如果你做完了，至少应该能清楚解释：

- `None` 和空字符串、`0` 的区别。
- 为什么 `input()` 后最好先做类型转换。
- 切片的起止范围规则。
- 解包赋值为什么比临时变量更自然。
- 列表推导式为什么更适合 Python。
- `list` 和 `dict` 的常用遍历方式。
