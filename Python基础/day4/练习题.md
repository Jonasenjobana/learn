# Day4 练习题：函数进阶与异常处理

## 题目 1：默认参数

写一个函数 `greet(name, message="你好")`。

要求：

- 不传 `message` 时输出默认问候
- 传了 `message` 时使用传入的内容

## 题目 2：任意数量求和

写一个函数 `add(*args)`。

要求：

- 可以接收任意多个数字
- 返回它们的总和

## 题目 3：关键字参数收集

写一个函数 `show_user(**kwargs)`。

要求：

- 接收任意用户信息
- 遍历并输出所有键和值

## 题目 4：解包传参

给定：

```python
nums = [3, 5]
```

写一个 `add(a, b)` 函数，然后用 `*nums` 调用它。

再给定：

```python
user = {"name": "小明", "age": 18}
```

写一个 `show(name, age)` 函数，然后用 `**user` 调用它。

## 题目 5：返回多个值

写一个函数 `get_min_max(numbers)`。

要求：

- 返回最小值和最大值
- 调用时用两个变量接收

## 题目 6：排序

给定：

```python
students = [
    {"name": "小明", "score": 88},
    {"name": "小红", "score": 95},
    {"name": "小刚", "score": 76},
]
```

要求：

- 按分数从低到高排序
- 使用 `lambda`

## 题目 7：输入异常处理

写一个程序，让用户输入年龄。

要求：

- 如果输入不是数字，提示“请输入有效数字”
- 如果输入正确，输出明年年龄

## 题目 8：除法异常处理

写一个函数 `divide(a, b)`。

要求：

- 如果 `b` 是 0，返回 `"不能除以 0"`
- 否则返回除法结果

## 题目 9：分数校验

写一个函数 `check_score(score)`。

要求：

- 如果分数小于 0 或大于 100，抛出 `ValueError`
- 否则返回分数本身

## 题目 10：综合练习

写一个函数 `create_student(name, score, **extra)`。

要求：

- 返回一个学生字典
- 必须包含 `name` 和 `score`
- `extra` 里的内容也要合并进去

示例：

```python
student = create_student("小明", 88, city="上海", age=18)
```

结果类似：

```python
{"name": "小明", "score": 88, "city": "上海", "age": 18}
```

## 检查标准

做完后你应该能解释：

- `*args` 和 `**kwargs` 的区别
- `*list` 和 `**dict` 在函数调用时分别做了什么
- 为什么默认参数不要随便用列表
- `try / except` 解决什么问题
- `raise` 是做什么的
