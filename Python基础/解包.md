# Python 解包

Python 里的“解包”，可以理解成把一个整体拆成多个部分，分别赋给多个变量。

它和 JavaScript 的解构赋值有点像，但 Python 里更常见、更自然的写法是：

- `a, b = ...`
- `*rest`
- `**kwargs`

## 1. 序列解包

最基础的形式，就是把列表、元组、字符串这类可迭代对象拆开赋值。

```python
a, b = [1, 2]
print(a)  # 1
print(b)  # 2
```

```python
x, y, z = (10, 20, 30)
print(x, y, z)
```

字符串也可以解包：

```python
p, q = "ab"
print(p)  # a
print(q)  # b
```

## 2. 交换变量

这是最常见的解包用法之一。

```python
a = 100
b = 200

a, b = b, a
print(a, b)
```

这种写法不需要临时变量，简洁又常见。

## 3. 星号解包 `*`

当你只想接住前几个或后几个值，剩下的交给一个变量时，就用 `*`。

```python
a, b, *rest = [1, 2, 3, 4, 5]
print(a)     # 1
print(b)     # 2
print(rest)  # [3, 4, 5]
```

`*` 也可以放在中间：

```python
first, *middle, last = [10, 20, 30, 40, 50]
print(first)   # 10
print(middle)  # [20, 30, 40]
print(last)    # 50
```

这在处理长度不固定的数据时特别有用。

## 4. 函数返回值解包

Python 函数如果返回多个值，通常本质上是返回一个元组，然后接收时直接解包。

```python
def get_user():
    return "小明", 18

name, age = get_user()
print(name, age)
```

这在后面写工具函数、数据处理函数时很常见。

## 5. 字典解包 `**`

字典解包常用于函数调用和合并字典。

### 5.1 函数调用

```python
def show_info(name, age):
    print(name, age)

info = {"name": "小明", "age": 18}
show_info(**info)
```

上面这句等价于：

```python
show_info(name="小明", age=18)
```

### 5.2 合并字典

```python
a = {"name": "小明"}
b = {"age": 18}

c = {**a, **b}
print(c)
```

输出：

```python
{'name': '小明', 'age': 18}
```

## 6. `*args` 和 `**kwargs`

这两个名字你会经常见到，它们也是解包思想的一部分。

### `*args`

表示接收任意多个位置参数。

```python
def add(*args):
    return sum(args)

print(add(1, 2, 3))
```

### `**kwargs`

表示接收任意多个关键字参数。

```python
def show(**kwargs):
    print(kwargs)

show(name="小明", age=18)
```

## 7. 解包的常见报错

### 数量不匹配

```python
a, b = [1, 2, 3]
```

会报错，因为左边只有两个变量，右边有三个值。

如果数量不固定，可以用 `*`：

```python
a, b, *rest = [1, 2, 3]
```

### 不够分

```python
x, y, z = [1, 2]
```

也会报错，因为右边不够三个变量分配。

## 8. 和 JavaScript 的关系

你可以把 Python 解包理解成“有点像 JS 解构赋值”，但 Python 更常见的是：

```python
a, b = values
```

以及：

```python
first, *middle, last = values
```

## 9. 一个综合例子

```python
def parse_user():
    return "Tom", 20, "Beijing"

name, age, city = parse_user()
print(name, age, city)

info = {"name": "Tom", "age": 20}

def show_info(**kwargs):
    print(kwargs)

show_info(**info)
```

## 10. 一句话总结

Python 解包就是把一个整体拆开，分别赋给多个变量。

常见形式是：

- `a, b = ...`
- `*rest`
- `**kwargs`

这在函数返回值、参数传递、数据处理里都非常常见。
