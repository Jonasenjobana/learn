# Day1 命令示例

## 查看 Docker

```bash
docker version
docker info
```

## 拉取并运行 nginx

```bash
docker pull nginx
docker images
docker run -d --name web -p 8080:80 nginx
```

浏览器访问：

```text
http://localhost:8080
```

## 查看状态和日志

```bash
docker ps
docker ps -a
docker logs web
docker logs -f web
```

## 进入容器

```bash
docker exec -it web sh
```

退出容器：

```bash
exit
```

## 停止和删除

```bash
docker stop web
docker rm web
```

## 清理提示

谨慎使用清理命令。学习阶段建议先明确目标，再删除：

```bash
docker ps -a
docker images
```
